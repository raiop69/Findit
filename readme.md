FINDIT – Lost & Found Portal Management System
Introduction
FINDIT is a web-based Lost & Found Portal Management System designed to help users report, search, and recover lost items efficiently. The system provides a centralized platform where users can report lost or found items, submit claims for ownership, and communicate through an admin-managed verification process.

Features
 User Features
  User Registration
  User Login
  Report Lost Items
  Report Found Items
  View Found Items
  View Reported Lost Items
  Submit Claims for Found Items
  Track Claim Status
 Admin Features
  View All Claim Requests
  Verify Ownership Claims
  Approve or Reject Claims
  Mark Items as Returned
